Web Framework

Web framework is a universal, reusable software environment to support the development of web applications, web services and websites. They aid in automating the overhead processes involved in web development, that is they can be considered as pre-built model for repetitive features associated in developing a website process.

    The benefits of using frameworks includes
  1. Easy Debugging
  2. Increased Code Efficiency
  3. Code Reusability
  4. Reinforced Security
    The apparent shortcomings includes
  1. Deters the language learning process
  2. Increased Website load time
  3. Limitation to customize
